### 本工程包含内容：
- MVP模式
- ViewPager+TabLayout实现多导航主页
- BaseActivity和BaseFragment的抽取及封装
- Dagger2实现依赖注入
- Retrofit+Rxjava+OkHttp实现数据请求
- 自定义组合控件实现类似携程主页的效果
